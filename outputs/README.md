# Outputs

This folder contains per-language outputs for the 7-language run.

Included for each language:

- `intervener_features.csv`
- `language_summary.csv`
- `zscore_results.csv`
- `ml_results.csv`
- `statistical_tests.csv`
- `condition_summary.csv`
- `kl_divergence.csv`
- `run_log.json`
- `figures/`

Not included in this GitHub-ready package:

- `distribution_data.csv`

Reason:

Some of the `distribution_data.csv` files are too large for a clean GitHub upload. They can be regenerated locally with the provided scripts.
