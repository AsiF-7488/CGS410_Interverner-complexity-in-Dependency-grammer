# Upload To GitHub

## Option 1: Create a new repository on GitHub, then push

```bash
cd CGS410--intervener-complexity-asif
git init
git branch -M main
git add .
git commit -m "Final CGS410 project deliverables"
git remote add origin <your-github-repo-url>
git push -u origin main
```

## Option 2: Upload the zip manually

Use:

`CGS410--intervener-complexity-asif.zip`
